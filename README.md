## 📌 Project Description & Purpose

A vanilla JavaScript app that fetches and displays data from the New York Times Books API. The app renders the NYT Bestsellers list with book covers, titles, and authors in a styled card layout.

## ✨ Features

- 📚 Fetches live book data from the NYT Books API
- 🖼️ Displays book covers, titles, and authors in a card layout
- 📅 Shows the date of the bestsellers list
- 🗞️ Styled to match the classic New York Times aesthetic
